This plugin displays functional overweight / BMI calculator for adults, kids and amputee. It can be used for english and german websites.


== Description ==
English:
_________
This plugin displays a functional overweight calculator / BMI calculator. It can be used to check the user's Body Mass Index.
This plugin can be used for english and german websites.

Deutsch:
_________
Dieses Plugin ist ein umfangreicher BMI-Rechner. Das BMI-Ergebnis wird genannt und es kann eine Erklaerung zur Gewichtseinstufung ueber eine Verlinkung abgerufen werden.
Das Plugin erkennt automatisch, ob ihr eine deutsche oder englische Wordpressversion benutzt. Es kann auf deutschen sowie englischen Wordpressseiten verwendet werden, da zweisprachig programmiert.


== Installation ==
English:
_________
1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Deutsch:
_________
1. Lade das gesamte Verzeichnis in dein Plugin Ordner "wp-content-plugins" hoch.
2. Aktiviere das Plugin unter "Pluging -> Installierte Plugins" im Adminbereich.
